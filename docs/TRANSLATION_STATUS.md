## Policy

- No machine translation without human review
- Official links are never translated (external German sources)
- Module page layouts inherit RTL from `html[dir=rtl]` already set by language switcher
- Translation notice shown in RTL languages: "Core interface translated. Full guide translations are in progress."

## How to contribute translations

1. Open an issue titled: `i18n: [Module name] — [Language]`
2. Copy the English content
3. Translate using a native speaker review
4. Submit a PR with the translated content
