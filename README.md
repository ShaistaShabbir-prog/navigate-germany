## What is Navigate Germany?

A unified mega-platform merging all Germany-related tools into one cohesive experience. Built for immigrants, expats, refugees, and new arrivals who need reliable, multilingual guidance on every aspect of life in Germany.

---

## ✨ Features

### 🗺️ 16 Bundesländer State Selector
Every German state with official government portal, Ausländerbehörde link, and local service portals. Click your state → get direct official links.

### 🌍 8 Languages
**English · Deutsch · العربية · Türkçe · اردو · Русский · Français · فارسی**

The entire interface translates — hero text, navigation, module titles. Language preference saved across sessions.

### 📦 12 Modules

| Module | Description |
|---|---|
| ⚖️ Legal Navigator + AI | AI letter reader, law database, rights, templates, chat |
| 📋 Bureaucracy Guide | Anmeldung, Krankenkasse, Steuer-ID, Bankkonto, Führerschein |
| 🏥 Doctor Finder | English/Arabic/Turkish/Urdu-speaking doctors, wait times |
| 💼 Jobs & Visa | Blue Card checker, 2026 thresholds, Chancenkarte, pathways |
| 🏘️ Housing | Tenant rights, Mietvertrag, rent disputes |
| 💶 Cost of Living | Budget calculator for 8 German cities |
| 🇩🇪 German Language | Phrases for Amt, doctors, work, transport, emergencies |
| 🚨 Emergency | All emergency numbers, urgent legal deadlines |
| ✉️ Letter Translator | AI translation of German official letters |
| 🏦 Banking | German bank accounts, Wise, Schufa |
| 📚 Integration | Integration courses, language schools, community |
| 👶 Family | Kindergeld, Kita, schools, family reunification |

### ⚖️ AI-Powered Legal Features (Claude claude-sonnet-4-20250514)
- Paste any German official letter → AI explains urgency, deadline, laws, actions
- Legal assistant chat — knows AufenthG, KSchG, BGB, SGB, AGG
- Draft reply letters with AI assistance
